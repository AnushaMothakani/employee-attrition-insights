# Employee Attrition Insights

## Project Overview

Employee Attrition Insights is a data analytics project that analyzes employee turnover patterns using Excel, MySQL, Python, and Power BI.

The objective of this project is to identify attrition trends, understand factors affecting employee retention, and provide business insights that can help organizations reduce employee turnover.

---

## Tools Used

* Excel – Data Cleaning & Preparation
* MySQL – Data Storage & Analysis
* Python – Data Analysis & Visualization
* Power BI – Interactive Dashboard

---

## Key Analysis

* Overall Attrition Rate
* Department-wise Attrition Analysis
* Job Role Analysis
* Salary Band Analysis
* Overtime Impact on Attrition
* Employee Tenure Analysis
* Attrition Cost Estimation

---

## Dashboard Features

* Employee Overview KPIs
* Attrition Rate Tracking
* Department Analysis
* Salary & Overtime Insights
* Financial Impact of Attrition
* Interactive Filters and Slicers

---

```text

   ↓
```

## 💡 Business Insights
